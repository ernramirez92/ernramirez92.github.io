Welcome to my first blog post! I’m excited to start blogging with Hugo and the Bearblog theme.

Why Hugo Bearblog?

The Bearblog theme is:

  • Fast - minimal JavaScript and CSS
  • Clean - focused on content
  • Simple - easy to customize

Getting Started

Creating new posts is easy. Just run:

hugo new content posts/my-new-post.md

Then edit the markdown file and set draft = false when you’re ready to publish.

Happy blogging!
